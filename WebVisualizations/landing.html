Latitude Analysis Dashboard

Weather Data Visualization Project

The purpose of this dashboard is to display weather data visualizations from the Python API homework assignment. This Home page provides an overview of the dashboard components. The dashboard includes four individual pages for each plot with the visualizations and explanations. The Comparision page contains all the visualizations on one page. Also included is a Data page with the data used to build the comparison plots.

See the navigation pane at the top of the page for links to each page of the dashboard. See the sidebar for preview images and links to each plot.
